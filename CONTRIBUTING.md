# Contributing

Thank you for your interest in CleanTextTools.

This repository exists primarily to provide public documentation, educational resources, and project information.

## Ways to Contribute

You can contribute by:

* Reporting documentation issues
* Suggesting improvements
* Identifying broken links
* Recommending new educational resources
* Sharing user feedback
* Reporting security concerns responsibly

## Before Opening an Issue

Please review existing issues to avoid duplicates.

When submitting feedback, include:

* A clear description of the problem
* Steps to reproduce if applicable
* Expected behavior
* Screenshots when relevant

## Feature Suggestions

Feature suggestions are welcome.

Please explain:

* The problem being solved
* Who benefits from the feature
* Why the feature is valuable

## Repository Scope

This repository does not contain:

* Proprietary backend code
* Internal infrastructure
* Private business strategy
* Deployment systems
* Analytics configurations
* Sensitive credentials

## Community Guidelines

Please be respectful and constructive.

We welcome feedback from users, developers, educators, writers, students, and professionals who work with AI-generated content.

## About CleanTextTools

CleanTextTools helps users clean AI-generated text for Google Docs, Microsoft Word, Gmail, Outlook, LinkedIn, and other platforms by removing formatting artifacts while preserving content meaning.
