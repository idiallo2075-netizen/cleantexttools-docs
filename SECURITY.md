# Security Policy

## Supported Versions

This repository contains public documentation and informational resources related to CleanTextTools.

## Reporting a Security Issue

If you discover a security vulnerability affecting CleanTextTools, please report it privately.

Do not publicly disclose vulnerabilities before they have been reviewed.

Please include:

* Description of the issue
* Steps to reproduce
* Potential impact
* Screenshots or supporting evidence if available

## Responsible Disclosure

We appreciate responsible disclosure and will review legitimate security reports as quickly as possible.

## Privacy and Data Handling

CleanTextTools is designed to minimize data exposure and support privacy-first workflows.

Where applicable, text processing is performed in the user's browser rather than being permanently stored.
